# Security Policy

## Supported Versions

Only the latest tagged release is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately.

**Do not open a public issue.**

Email: security@sha888.dev (placeholder — update with real contact)

Please include:
- A description of the vulnerability
- Steps to reproduce (if applicable)
- Your assessment of severity

We aim to acknowledge receipt within 48 hours and provide a remediation timeline within 7 days.
